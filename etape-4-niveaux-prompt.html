4. Comparaison des niveaux de prompt

Même scénario, même IA — seul le niveau de précision du prompt change le résultat.

Scénario : Je raconte mes compétences  |  Objectif : Je peux raconter mes compétences techniques  |  Champ d'action : Recherche d'emploi

Niveau 1 — Vide

Aucun cadre. Résultat générique, imprévisible et inutilisable.

"J'ai besoin d'un cours FIDE sur les compétences. Fais quelque chose."

Manques :

  • Objectif communicatif non défini
  • Niveau CECR absent
  • Aucune tâche finale observable
  • Pas d'ancrage FIDE ni contexte suisse

Niveau 2 — Flou

Le sujet est nommé, mais aucun paramètre pédagogique n'est précisé.

"Fais-moi un cours FIDE sur les compétences."

Manques :

  • Niveau CECR absent
  • Champ d'action FIDE non précisé
  • Aucune tâche finale observable
  • Résultat imprévisible, souvent trop générique

Niveau 3 — Structure simple

Quelques mots-clés présents, mais pas assez de précision pour guider l'IA.

"Crée une séquence FIDE A1 recherche d'emploi pour parler des compétences."

Manques :

  • Scénario trop général
  • Absence de déroulé par étapes
  • Aucune consigne d'évaluation formative
  • Supports et différenciation non demandés

Niveau 4 — Formateur sérieux

Bon cadre de base — mais les éléments FIDE essentiels sont encore absents.

"Séquence FIDE A1 (90 min), champ recherche d'emploi, scénario : je raconte mes compétences."

Manques :

  • Dialogues modèles non exigés
  • Ressources langagières non détaillées
  • Critères observables d'évaluation absents
  • Connaissances socioculturelles peu explicites
👍  Même scénario, même IA. La différence entre niveau 0 et niveau 5, c'est votre précision — pas l'outil.
